# RSP

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 14.2.5.

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Typescript
- Javascript
- Angular CLI

## Features
- Create an account
  - Add home post
  - View profile
    - Edit Profile
  - View all posts location
    - View post
  - View watchlist
  - View post
    - Add post to watchlist
    - remove post from watchlist
    - Make reservation
    - View profile host
    - Contact host
